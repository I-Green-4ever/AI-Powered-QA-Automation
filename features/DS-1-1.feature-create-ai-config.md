## Selected gap (one flow)
**Flow:** Expand and collapse AI Generation Config in the New Program modal  
**Why this one:** DS-1 exercises only Program Name and Description, but the live create dialog exposes a collapsible AI config section with six additional controls — a distinct UI region with zero spec coverage and no POM support, so regressions there would go unnoticed.
